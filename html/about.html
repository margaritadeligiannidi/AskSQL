About AskSQL

AskSQL is a web application that allows users to interact with relational databases using either SQL queries or natural language questions.

The system simplifies database access by translating natural language requests into SQL queries, enabling users to retrieve information without requiring advanced SQL knowledge. It also supports database connection management, query execution, result visualization, CSV export, and voice-based query input.

Key Features

Natural Language to SQL
SQL Query Execution
Multiple AI Providers
Connection Management
Result Visualization
CSV Export
Voice Input
Secure Authentication

Academic Project

This web application was created for the Department of Information and Electronic Engineering - International Hellenic University as a thesis by Margarita Deligiannidi.
